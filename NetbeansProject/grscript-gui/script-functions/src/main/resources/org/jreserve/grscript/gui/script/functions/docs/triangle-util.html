Triangle Utilities

GRScript provides utility methods to make working with triangular data easier.

  • cummulate(double[]): cummulates the given array.
  • cummulate(double[][]): cummulates the given matrix. The first index is assumed to be the accident period (0-based), the second index is assumed to be the development period (0-based).
  • decummulate(double[]): decummulates the given array.
  • cummulate(double[][]): decummulates the given matrix. The first index is assumed to be the accident period (0-based), the second index is assumed to be the development period (0-based).
  • copy(double[]) / copy(double[][]): copies the given vector or matrix.
  • add(double[] a, double[] b), add(double[][] a, double[][] b): adds the two vectors/matrices. One can simply use a+b also.
  • subtract(double[] a, double[] b), subtract(double[][] a, double[][] b): subtracts the two vectors/matrices. One can simply use a-b also.
  • divide(double[] a, double[] b), divide(double[][] a, double[][] b): divides the two vectors/matrices. One can simply use a/b also.
  • multiply(double[] a, double[] b), multiply(double[][] a, double[][] b): multiplies the two vectors/matrices. One can simply use a*b also.
