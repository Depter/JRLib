Link-Ratios

GRScripts provides several methods to create link-ratios based on FactorTriangles.

  • linkRatio(double[][]): creates a LinkRatio from the given triangle, using the weighted-average method for all development periopds.
  • linkRatio(ClaimTriangle): creates a LinkRatio from the given triangle, using the weighted-average method for all development periopds.
  • linkRatio(FactorTriangle): creates a LinkRatio from the given factors, using the weighted-average method for all development periopds.

Builder Syntax

The builder syntax allows the use of different methods for different development periods. 

data = cummulate(apcPaid())
lr = linkRatios(data) {
    average(0)          //-> use simple average for the first column
    mack(1)             //-> use mack (linear-regression) method
    max(2)              //-> use the maximum of the individual factors
    min(3)              //-> use the minimum of the individual factors
    weightedAverage(4)  //-> use the weighted average
    fixed(5, 1.05)      //-> use a constant value for the 6th column
}

//In addition multiple indices can be specified
lr = linkRatios(data) {
    average(0..2)          //-> use simple average for the first 3 columns
    mack(3, 5)             //-> use mack (linear-regression) method for column 4 and 6
    fixed(5:1.05, 7:1.01)  //-> for fixed, the map-syntax can be used
}
