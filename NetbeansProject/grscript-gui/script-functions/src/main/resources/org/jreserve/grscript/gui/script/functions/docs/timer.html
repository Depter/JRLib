Timer

The timer functionality provides a convenient way to measure the duration of some task within a script

  • startTimer(String task): Starts a timer for the given task.
  • stopTimer(String task): Stops the timer associated with the given task, retuns the duration of the rask in miliseconds and clears the timer.
  • finnishTimer(String task): Stops the timer associated with the given task, prints the duration of the task to the output and clears the timer.

Examples

startTimer("Task_1")
startTimer("Task_2")

//Stop for about 1 second
Thread.sleep(1000)
duration = stopTimer("Task_1")

println "Task_1: ${duration} miliseconds"

//Stop for about 1 second
Thread.sleep(2000)

finnishTimer("Task_2")
