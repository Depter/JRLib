Printing Data

As default, for most of the differnet data types, such as triangles, vectors, histogram data, two print data methods are provided to conveniently print the values of the object in a user-friendly format.

Methods

The basic functions are:
  • printData(Object): prints the given object.
  • printData(String, Object): prints the given title, followed by the given object in a new line.

In addition to printing data, one can influence how the values are formatted. The basic format for numbers is #.##. Use to setNumberFormat(String) method to change the default format. To change the locale, use setLocale(String)

Examples

//Set the default number format to 3 decimals
setNumberFormat "0.000"

//Set the locale to english
setLocale "en"
