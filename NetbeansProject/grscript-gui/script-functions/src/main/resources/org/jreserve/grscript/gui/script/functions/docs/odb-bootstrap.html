ODP-Bootstrap

The ODP bootstrap allows to bootstrap chain-ladder reserves, asssuming an overdispersed poisson model. The bootstrap requires 3 input

  • Residuals
  • Random Generator
  • Process simulator

Random Generator

A random generator is needed to reshuffle the residuals from the residual triangle, and to symulate the process variance. In the current version only the default Java random generator is supported, but if it is desired, additional random generators can be plugged in, by implementing the org.jreserve.jrlib.util.random.Random interface.

Per default the bootstrap uses an instance of the Java random generator. To plug-in other implementations or to set the seed, use:

  • random(Random rnd): where rnd is a custom implementation for the org.jreserve.jrlib.util.random.Random interface.
  • random(String type, long seed): where type must be "Java".
  • Use builder syntax: 
        random {
        type "Java"
        seed 1000
    }
